# Technology Stack

**Analysis Date:** 2026-04-14

## Languages

**Primary:**
- JavaScript (ES6+) - All source code in `src/` and plugin entry point

**File Extensions:**
- `.js` - Source files and tests

## Runtime

**Environment:**
- Obsidian (Electron-based) - Plugin runs within Obsidian.md desktop/web application
- Minimum Obsidian version: 0.12.12 (from `manifest.json`)

**Package Manager:**
- pnpm (lockfile: `pnpm-lock.yaml`)
- Node.js - For development and build tooling

## Frameworks

**Core:**
- Obsidian API (master branch from GitHub) - Plugin framework for extending Obsidian.md
  - Imported from `"obsidian"` package

**Utilities:**
- obsidian-daily-notes-interface ^0.9.2 - Helper library for daily notes functionality
  - Provides: `getDailyNoteSettings()`, `getAllDailyNotes()`, `getDailyNote()`

**Testing:**
- Vitest ^3.1.2 - Unit test framework
  - Config: Default (no vitest.config.js file in repo)
  - Run command: `npm test` or `pnpm test`

**Build/Dev:**
- Rollup 2.32.1 - Module bundler
  - Config: `rollup.config.js`
  - Entry point: `src/index.js`
  - Output: `main.js` (CommonJS format)
  - Plugins:
    - @rollup/plugin-node-resolve ^9.0.0 - Resolves npm dependencies
    - @rollup/plugin-commonjs ^15.1.0 - Converts CommonJS to ES modules

**Formatting:**
- Prettier ^2.8.1 - Code formatter
  - Run command: `npm run format` or `pnpm format`

## Key Dependencies

**Critical:**
- obsidian (master/latest) - Required peer dependency; provides `Plugin`, `Notice`, `PluginSettingTab` classes
- obsidian-daily-notes-interface ^0.9.2 - Essential for daily note operations

**Build Only (devDependencies):**
- rollup ^2.32.1 - Bundles plugin code for distribution
- @rollup/plugin-node-resolve ^9.0.0 - Node module resolution
- @rollup/plugin-commonjs ^15.1.0 - CommonJS interoperability
- prettier ^2.8.1 - Code formatting
- vitest ^3.1.2 - Test execution

## Configuration

**Build Configuration:**
- `rollup.config.js` - Defines bundling process
  - Marks `obsidian` as external (not bundled)
  - Outputs to `main.js` with inline sourcemaps (production excludes sources)
  - Uses CommonJS format for Obsidian plugin compatibility
  - Browser resolution enabled for node dependencies

**Package Configuration:**
- `package.json` - Project metadata and scripts
  - Main entry: `main.js` (bundled output)
  - Scripts:
    - `dev`: `rollup -c -w` (watch mode)
    - `build`: `rollup -c --environment BUILD:production`
    - `test`: `vitest`
    - `format`: `prettier --write .`

**Manifest Configuration:**
- `manifest.json` - Obsidian plugin metadata
  - id: obsidian-rollover-daily-todos
  - version: 1.2.0
  - minAppVersion: 0.12.12
  - isDesktopOnly: false (runs on desktop and web)

**Version Management:**
- `versions.json` - Version history tracking
